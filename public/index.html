Myanmar Identity API

Validate and parse Myanmar National Registration Cards with our comprehensive API. Test NRC formats, get township details, and build valid NRCs step by step.

Launch playground
Node + Express Myanmar NRC Validation API

Quick Start

npm install
npm start
# or live reload
npm run dev

API Endpoints

  • GET /api/nrc-types
  • GET /api/states
  • GET /api/townships
  • GET /api/states/:stateId/townships
  • GET /api/states/code/:code
  • GET /api/townships/code/:code
  • GET /api/states/number/:number/townships
  • POST /api/nrc/parse
